Detailed Description

Generic parameter sub-class.

This class will store an parameter inside the parameter list. The parameter necessarily has a type and value. In case of strings the value is dinamically allocated.

If no type is assigned, objects will be initialized as Parameter:Unitialized. After a type is assigned, the object will remain as the same time throughout it's entire lifetime. In case a method related to a type other than the object's, an OException will be thrown.
